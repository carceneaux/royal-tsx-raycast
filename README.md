# Royal TSX - Raycast Extension

Search and connect to your [Royal TSX](https://royalapps.com/ts/mac/features) connections directly from Raycast.

## Requirements

- [Royal TSX](https://royalapps.com/ts/mac/features) must be installed and running with a document open.

## Commands

- **Connection List**: Search all connections from your open Royal TSX document and connect to them. Supports ad hoc connections when no match is found.

## Features

**Search your connections**
- Instantly search connections by name using multi-word filtering.
- Connections are sorted alphabetically for easy browsing.

**Recent connections**
- The last 10 connections you used are surfaced in a dedicated **Recent** section at the top of the list.

**Ad hoc connections**
- When your search text doesn't match any existing connection, an **Ad Hoc Connection** item appears. Select it to connect directly to any hostname without needing a saved connection.

**Error handling**
- If Royal TSX is not running when the extension is invoked, a clear message is displayed instead of a silent failure.
